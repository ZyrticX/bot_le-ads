## ⚡ התקנה מהירה (15 דקות)

### שלב 1: הכנה
אתה צריך:
- ✅ Python 3.11.9 (יש לך ✅)
- ⏳ חשבון Telegram
- ⏳ חשבון Airtable (חינמי)

### שלב 2: יצירת בוט
1. פתח [@BotFather](https://t.me/BotFather) בטלגרם
2. שלח: `/newbot`
3. תן שם ו-username
4. **שמור את ה-Token!**

### שלב 3: הקמת Airtable
1. היכנס ל-[airtable.com](https://airtable.com)
2. צור Base בשם `Leads CRM`
3. צור 3 טבלאות: `Deals`, `Clients`, `Payments`
4. קבל **API Key** ו-**Base ID**

📖 **מדריך מפורט:** [AIRTABLE_SETUP.md](AIRTABLE_SETUP.md)

### שלב 4: התקנה במחשב

**אוטומטי (מומלץ):**
```powershell
.\setup_windows.ps1
```

**או ידני:**
```powershell
# 1. צור סביבה וירטואלית
python -m venv venv

# 2. הפעל
venv\Scripts\Activate.ps1

# 3. התקן
pip install -r requirements.txt

# 4. ערוך הגדרות
notepad .env
```

### שלב 5: מילוי הגדרות
פתח את `.env` ומלא:
```env
TELEGRAM_BOT_TOKEN=<מ-BotFather>
AIRTABLE_API_KEY=<מAirtable>
AIRTABLE_BASE_ID=<מAirtable>
```

### שלב 6: הפעלה!
```powershell
python bot.py
```

אמור לראות:
```
🚀 Bot is starting...
```
